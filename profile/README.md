# CANGO2 (Robotic Guide Dog)
Kwangwoon University Department of Robotics Capston project

## Project Overview
Autonomous Guide Robot for the Visually Impaired.

Main focus areas include:
- Autonomous navigation
- Sensor integration
- LiDAR-based perception
- Robot control systems
- Real-world robotics applications
